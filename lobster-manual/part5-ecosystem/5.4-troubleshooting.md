## 5.4 故障排查：常见问题

**Q: 遇到问题怎么办？有哪些常见故障？A: 80% 的问题可以自己解决，按这个清单查。**

---

### 【先说结论】

```
故障排查顺序：

第 1 步：问龙虾自己（80% 能解决）
第 2 步：查这个清单（15% 能解决）
第 3 步：问社群（4% 能解决）
第 4 步：找客服（1% 需要）

核心：
大部分问题不难，
按步骤查就行。
```

---

### 【安装问题】

**问题 1：安装失败**
```
现象：
- 点击安装没反应
- 报错 XXX

原因：
- 网络问题
- 权限问题
- 系统不兼容

解决：
1. 检查网络（能正常上网吗？）
2. 检查权限（管理员权限）
3. 检查系统（Windows 10+/macOS 11+）
4. 重启电脑再试
5. 还不行问龙虾："安装失败怎么办？"
```

**问题 2：部署卡住**
```
现象：
- 部署进度卡住
- 一直加载中

原因：
- 网络慢
- 模型下载大
- 服务器忙

解决：
1. 检查网络速度
2. 暂停重新开始
3. 换个时间（避开高峰）
4. 用国内镜像源
5. 问龙虾："部署卡住怎么办？"
```

---

### 【连接问题】

**问题 3：飞书连不上**
```
现象：
- 飞书授权失败
- 消息发不出去

原因：
- 授权过期
- 飞书版本低
- 网络问题

解决：
1. 重新授权（设置 → 飞书 → 重新授权）
2. 更新飞书（最新版）
3. 检查网络
4. 重启龙虾
5. 问龙虾："飞书连不上怎么办？"
```

**问题 4：消息不同步**
```
现象：
- 手机电脑消息不一致
- 延迟

原因：
- 网络延迟
- 同步设置问题

解决：
1. 检查网络
2. 设置 → 同步 → 开启实时同步
3. 重启飞书
4. 问龙虾："消息不同步怎么办？"
```

---

### 【性能问题】

**问题 5：运行卡顿】
```
现象：
- 响应慢
- 电脑卡

原因：
- 内存不足
- 模型太大
- 其他程序占用

解决：
1. 关闭其他程序
2. 换小模型（Gemma 4 Nano）
3. 加内存（8G→16G）
4. 用云端 API（不占本地）
5. 问龙虾："运行卡顿怎么办？"
```

**问题 6：内存不足】
```
现象：
- 报错"内存不足"
- 程序崩溃

原因：
- 内存太小
- 模型太大
- 同时运行太多

解决：
1. 关闭其他程序
2. 换小模型
3. 加内存（¥200-300）
4. 用云端版
5. 问龙虾："内存不足怎么办？"
```

---

### 【功能问题】

**问题 7：技能不能用】
```
现象：
- 技能报错
- 技能不执行

原因：
- 技能未安装
- 技能版本旧
- 配置问题

解决：
1. 检查技能是否安装
2. 更新技能（设置 → 技能 → 更新）
3. 重新配置技能
4. 问龙虾："XXX 技能不能用怎么办？"
```

**问题 8：记忆丢失】
```
现象：
- 龙虾不记得之前的信息
- 记忆清空

原因：
- 记忆文件损坏
- 清理设置
- 版本升级

解决：
1. 检查记忆文件（~/.lobster/data/memory.db）
2. 关闭自动清理
3. 恢复备份
4. 问龙虾："记忆丢失怎么办？"
```

---

### 【账号问题】

**问题 9：登录失败】
```
现象：
- 登录报错
- 账号不存在

原因：
- 密码错误
- 账号未注册
- 服务器问题

解决：
1. 检查密码（大小写）
2. 确认账号已注册
3. 重置密码
4. 问龙虾："登录失败怎么办？"
```

**问题 10：权限不足】
```
现象：
- 报错"权限不足"
- 某些功能不能用

原因：
- 账号权限低
- 未付费
- 企业限制

解决：
1. 检查账号权限
2. 确认付费状态
3. 联系管理员（企业版）
4. 问龙虾："权限不足怎么办？"
```
