## 2.1 能干什么活？（核心能力）

**Q: 龙虾到底能帮我干什么活？**

**A: 能帮你干三类活：查、写、做。**

---

### 【第一类：查资料】

**场景：** 你想知道某个信息，但懒得搜/搜不到/看不懂

**龙虾能做的：**
- ✅ 查行业报告（"帮我找 2026 年 AI 行业报告"）
- ✅ 查政策法规（"上海居住证积分政策是什么？"）
- ✅ 查竞品信息（"淘宝上类似我的产品卖多少钱？"）
- ✅ 查学习资料（"帮我找小升初数学真题"）
- ✅ 查新闻热点（"今天 AI 行业有什么大事？"）

**和百度/谷歌的区别：**
- ❌ 搜索引擎：给你 10 个链接，你自己看
- ✅ 龙虾：看完总结好，直接给你答案

**真实案例：**
```
用户："帮我查一下上海公办初中入学条件"
龙虾：（30 秒后）
"已查好，非沪籍需要：
1. 居住证（父母一方）
2. 积分≥120 分
3. 社保连续缴纳
4. 实际居住证明
具体政策文件链接：xxx"
```

---

### 【第二类：写内容】

**场景：** 你需要写东西，但没时间/不会写/写不好

**龙虾能做的：**
- ✅ 写文案（商品详情/朋友圈/小红书）
- ✅ 写报告（日报/周报/会议纪要）
- ✅ 写邮件（客户沟通/合作邀约）
- ✅ 写方案（活动策划/营销计划）
- ✅ 写脚本（短视频/直播话术）

**和豆包/ChatGPT 的区别：**
- ❌ 豆包：聊天为主，写完就结束
- ✅ 龙虾：写完还能帮你发/帮你改/帮你优化

**真实案例：**
```
用户："帮我写一个淘宝商品标题，卖儿童学习桌"
龙虾：（10 秒后）
"已写好 5 个版本：
1. 【护脊神器】儿童学习桌可升降...
2. 小学生写字桌实木护眼...
3. ...
推荐第 2 个，点击率高"
```

---

### 【第三类：做任务】

**场景：** 你有重复性工作，想自动化

**龙虾能做的：**
- ✅ 整理数据（Excel 汇总/格式转换）
- ✅ 文件管理（分类/重命名/备份）
- ✅ 定时提醒（会议/缴费/生日）
- ✅ 多平台操作（同步发布/批量处理）
- ✅ 流程自动化（订单处理/客户跟进）

**和传统软件的区别：**
- ❌ 传统软件：你要点很多按钮
- ✅ 龙虾：你说一句话，它自动完成

**真实案例：**
```
用户："把这 100 个订单信息整理成 Excel"
龙虾：（2 分钟后）
"已整理好，包含：
- 订单号
- 客户姓名
- 商品名称
- 金额
- 发货状态
文件已保存到：xxx.xlsx"
```

### 【能力边界】

**龙虾做不到的：**
- ❌ 需要人类判断的事（审美/情感/道德）
- ❌ 需要物理操作的事（做饭/打扫/开车）
- ❌ 需要专业资质的事（看病/打官司/审计）
- ❌ 100% 准确的事（可能出错，需要检查）

**正确心态：**
- ✅ 龙虾是"助理"，不是"超人"
- ✅ 重要事情要检查，不要完全依赖
- ✅ 让它做重复活，你做创造性活

## 2.2 AI 能自动赚钱吗？（商业模式澄清）

**Q: 装了龙虾就能自动赚钱吗？**

**A: 不能。龙虾是"帮你赚钱的工具"，不是"印钞机"。**

---

### 【先说结论】

```
❌ 不能：
- 躺着不动，钱自动进账
- 什么都不做，AI 帮你赚钱
- 今天装，明天就发财

✅ 能：
- 帮你提高效率（省时间=省钱）
- 帮你做更多活（产能提升=收入增加）
- 帮你减少错误（少损失=变相赚钱）
```

---

### 【为什么有人宣传"AI 赚钱"？】

**三种情况：**

**1. 卖课的（99%）**
```
话术："用 AI 月入 10 万"
真相：赚的是你的学费，不是 AI 赚的钱
目的：让你买课/买服务
```

**2. 真赚到钱的（1%）**
```
案例：电商卖家用 AI 写文案，销量提升 30%
真相：AI 是工具，核心还是产品好 + 运营好
关键：AI 放大能力，不是替代能力
```

**3. 误解的**
```
用户："AI 帮我写文章，流量涨了"
真相：AI 写得好 + 你选题好 + 平台推荐
误解：以为是 AI 自己赚钱
```

---

### 【龙虾怎么帮你"赚钱"？】

**方式 1：省时间 = 省钱**
```
场景：你每天花 2 小时写日报
龙虾：帮你写，10 分钟搞定
价值：每天省 1 小时 50 分钟
换算：1 小时×22 天×时薪 50 元 = 1100 元/月
```

**方式 2：提效率 = 多接单**
```
场景：你接文案兼职，一天写 3 篇
龙虾：帮你写初稿，你修改
价值：一天能写 8 篇
换算：多 5 篇×100 元/篇×22 天 = 11000 元/月
```

**方式 3：少出错 = 少损失**
```
场景：订单信息手动录入，每月错 5 单
龙虾：自动整理，0 错误
价值：少损失 5 单×200 元/单 = 1000 元/月
```

**方式 4：做更好 = 卖更高**
```
场景：淘宝店商品文案普通，转化率 2%
龙虾：优化文案，转化率 3.5%
价值：多 1.5% 转化×10 万流量×100 元客单 = 15000 元/月
```

## 2.3 我会失业吗？（AI 替代性）

**Q: AI 这么厉害，我会不会失业？**

**A: 不会"被 AI 取代"，但会"被会用 AI 的人取代"。**

---

### 【先说结论】

```
❌ 不会：
- AI 完全取代你的工作
- 明天起床就没工作了
- 所有岗位都被 AI 干掉

✅ 会：
- 部分重复工作被 AI 替代
- 不会用 AI 的人竞争力下降
- 工作内容由"做事"变"管 AI"

核心：
不是"AI 取代人"，是"会用 AI 的人取代不会用的人"。
```

---

### 【哪些工作容易被替代？】

**高风险（重复性高/规则明确）：**
```
- 数据录入员（AI 自动整理）
- 基础客服（AI 自动回复）
- 简单翻译（AI 翻译准确率高）
- 基础文案（AI 写稿快）
- 电话销售（AI 外呼）
```

**中风险（部分可替代）：**
```
- 会计（AI 做账，人审核）
- 设计师（AI 出图，人选稿）
- 程序员（AI 写代码，人架构）
- 编辑（AI 初稿，人修改）
- 运营（AI 分析，人决策）
```

**低风险（需要人类判断）：**
```
- 管理者（决策/协调/激励）
- 创意工作者（创意/审美/情感）
- 专业服务（医生/律师/咨询师）
- 销售（关系建立/谈判）
- 手工艺人（精细操作/个性化）
```

---

### 【真实案例】

**案例 1：客服（被替代）**
```
用 AI 前：
- 10 个客服
- 每天回复 500 个咨询
- 人力成本 5 万/月

用 AI 后：
- 2 个客服 + AI
- 每天回复 800 个咨询
- 人力成本 1 万/月

结果：
8 个客服失业/转岗
2 个客服升级为"AI 训练师"
```

**案例 2：文案（升级）**
```
用 AI 前：
- 每天写 3 篇文案
- 每篇 2 小时
- 月薪 8000

用 AI 后：
- 每天写 8 篇（AI 初稿 + 人修改）
- 每篇 30 分钟
- 月薪 15000

结果：
没失业，反而升职加薪
从"写手"变成"内容主管"
```

**案例 3：设计师（共存）**
```
用 AI 前：
- 每天做 2 个设计稿
- 每个 3 小时
- 客户满意度 80%

用 AI 后：
- 每天做 5 个（AI 出图 + 人优化）
- 每个 1 小时
- 客户满意度 95%

结果：
没失业，效率提升 150%
从"美工"变成"创意总监"
```

---

### 【如何避免被替代？】

**策略 1：学会用 AI**
```
- 今天就开始用（不要等）
- 用它做重复工作（省时间）
- 用省下的时间学新技能（提升）
```

**策略 2：做 AI 做不了的事**
```
- 创意（AI 没有真正创意）
- 情感（AI 没有真实情感）
- 判断（AI 没有道德判断）
- 关系（AI 建立不了信任）
```

**策略 3：成为"管 AI"的人**
```
- 学会给 AI 下指令（提示词）
- 学会检查 AI 输出（质量控制）
- 学会优化 AI 流程（效率提升）
```

---

### 【未来工作形态】

```
现在：
人 → 做事 → 交付

未来：
人 → 管 AI → AI 做事 → 人检查 → 交付

变化：
- 从"执行者"变成"管理者"
- 从"做事"变成"决策"
- 从"重复劳动"变成"创造性工作"
```

---

### 【一句话总结】

```
AI 不会取代你，
但会用 AI 的人会取代你。

就像：
- 汽车没有取代司机，但取代了马车夫
- 电脑没有取代会计，但取代了算盘手
- AI 不会取代你，但会改变你的工作方式

核心：
不要问"AI 会不会取代我"，
要问"我怎么用 AI 让自己更值钱"。
```

## 2.4 记忆能力：AI 记得住我吗？

**Q: AI 能记住我的信息吗？下次聊天还记得吗？**

**A: 看情况。普通 AI 记不住，龙虾能记住。**

---

### 【先说结论】

```
❌ 普通 AI（豆包/ChatGPT 网页版）：
- 每次对话都是新的
- 关掉窗口就忘了
- 你要重复说同样的话

✅ 龙虾（本地部署 + 记忆功能）：
- 记住你的偏好
- 记住历史对话
- 记住家庭/工作信息
- 越用越懂你
```

---

### 【为什么普通 AI 记不住？】

**原因 1：隐私保护**
```
- 不存储你的数据
- 每次对话独立
- 安全，但不方便
```

**原因 2：技术限制**
```
- 云端对话量大
- 存储成本高
- 无法为每个人单独记忆
```

**原因 3：产品设计**
```
- 定位是"聊天工具"
- 不是"个人助理"
- 用完即走
```

---

### 【真实痛点】

**场景 1：重复介绍自己**
```
用户："我叫张三，在上海做电商"
（第二天）
用户："帮我写个文案"
AI："好的，请问您是做什么行业的？"
用户："...我不是昨天说了吗"
```

**场景 2：偏好记不住**
```
用户："用简洁风格写"
（下次）
用户："写个报告"
AI：（写了 5000 字长篇）
用户："我不是说要简洁吗？！"
```

**场景 3：家庭信息重复说**
```
用户："我女儿叫 Zoe，五年级"
（下周）
用户："帮我找出题"
AI："请问您孩子几年级？"
用户："..."
```

---

### 【龙虾怎么记住你？】

**方式 1：本地存储**
```
- 数据存在你电脑/手机上
- 不上传云端
- 只有你能看
- 安全 + 持久
```

**方式 2：记忆文件**
```
- 自动记录关键信息
- 姓名/职业/偏好/家庭
- 每次对话自动更新
- 像"个人档案"
```

**方式 3：长期记忆**
```
- 记住历史对话
- 记住任务进度
- 记住文件位置
- 记住使用习惯
```

### 【隐私安全吗？】

**用户担心：**
```
"AI 记住我这么多信息，安全吗？"
"会不会泄露？"
"谁能看？"
```

**龙虾的方案：**
```
✅ 数据本地存储（在你电脑/手机）
✅ 不上传云端（没人能看）
✅ 你可以删除（随时清空）

```

**对比云端 AI：**
```
❌ 云端 AI：数据在服务器上
   - 公司能看
   - 可能被黑客攻击
   - 你无法控制

✅ 本地龙虾：数据在你手里
   - 只有你能看
   - 黑客攻不破（不联网）
   - 你完全控制
```

```

❌ 依赖网络（没网不能用）
```

```
- 首次：¥0
```

**适合：**

```
✅ 完全不懂技术
✅ 轻度使用
✅ 老电脑用户

```

## 4.6 提示词技巧：让 AI 输出翻倍

**Q: 怎么跟龙虾说话，它才能更好理解我的意思？**

**A: 记住一个核心：把它当真人，说人话。**

---

### 【先说结论】

```
❌ 错误方式：
- 像下命令（"写文案"）
- 太简短（"总结"）
- 没上下文（"这个怎么样"）

✅ 正确方式：
- 像跟助理说话（"帮我写个文案"）
- 给背景信息（"淘宝卖儿童学习桌"）
- 说清楚要求（"简洁风格，200 字以内"）

核心：
把它当真人助理，
你说得越清楚，
它做得越好。
```

---

### 【基础公式】

**好提示词 = 角色 + 任务 + 背景 + 要求**

**示例 1：写文案**
```
❌ 差："写文案"
✅ 好："你是淘宝文案专家，帮我写个儿童学习桌的商品标题，
       卖点是护脊/可升降/实木，
       风格简洁，20 字以内，写 5 个版本"
```

**示例 2：查资料**
```
❌ 差："查政策"
✅ 好："你是政策研究员，帮我查上海公办初中入学条件，
       我是非沪籍家庭，居住证积分达标，
       需要具体条件和材料清单，
       附上官方文件链接"
```

**示例 3：整理数据**
```
❌ 差："整理 Excel"
✅ 好："你是数据分析师，把这 100 个订单整理成 Excel，
       包含：订单号/客户姓名/商品/金额/状态，
       按金额排序，
       标出超过 1000 元的订单"
```

---

### 【进阶技巧】

**技巧 1：给角色**
```
"你是 XXX 专家..."

示例：
- "你是资深文案..."
- "你是数据分析师..."
- "你是教育顾问..."
- "你是法律顾问..."

作用：让它进入角色，输出更专业。
```

**技巧 2：给背景**
```
"背景是 XXX..."

示例：
- "我是淘宝卖家，卖儿童学习桌"
- "我女儿五年级，数学好，英语弱"
- "我们公司做 AI 培训，面向小白"

作用：让它了解你，输出更精准。
```

**技巧 3：给示例**
```
"像这样..."

示例：
- "标题风格像这个：【护脊神器】儿童学习桌..."
- "报告格式参考附件"
- "用上次那种简洁风格"

作用：让它有参照，输出更符合预期。
```

**技巧 4：给限制**
```
"要求 XXX..."

示例：
- "200 字以内"
- "用表格形式"
- "分 3 个部分"
- "不要专业术语"

作用：让它有边界，输出更可控。
```

```
✅ 建议：
```
