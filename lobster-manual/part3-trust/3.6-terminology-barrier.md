## 3.6 理解门槛：看不懂技术术语

**Q: 全是技术术语，看不懂怎么办？**

**A: 不用懂，这本手册就是说人话的。**

---

### 【先说结论】

```
❌ 不用懂：
- 不用懂"部署"是什么
- 不用懂"API"怎么工作
- 不用懂"Token"怎么计算

✅ 只需要懂：
- 怎么用（发消息）
- 能干什么（查/写/做）
- 遇到问题怎么办（问 AI）

核心：
就像用微信，
不用懂 4G/5G。
```

---

### 【常见术语翻译】

**部署 = 安装**
```
术语："部署 OpenClaw"
人话："安装龙虾"

就像：
- "部署微信" = "安装微信"
- "部署抖音" = "安装抖音"
```

**API = 连接**
```
术语："配置 API Key"
人话："连接 AI 大脑"

就像：
- "连接 WiFi"
- "绑定银行卡"
```

**Token = 字数**
```
术语："消耗 1000 Token"
人话："用了 1000 字"

就像：
- "手机流量"
- "话费分钟数"
```

**模型 = 大脑**
```
术语："Gemma 4 模型"
人话："AI 大脑"

就像：
- "手机芯片"
- "电脑 CPU"
```

**本地部署 = 装自己电脑**
```
术语："本地部署"
人话："装在自己电脑上"

就像：
- "本地音乐" = "电脑里的音乐"
- "本地文件" = "电脑里的文件"
```

**云端 = 网上**
```
术语："云端存储"
人话："存在网上"

就像：
- "云盘" = "网上硬盘"
- "云备份" = "网上备份"
```

**技能 = 功能**
```
术语："安装技能"
人话："安装功能"

就像：
- "微信小程序"
- "手机 APP"
```

**提示词 = 你说的话**
```
术语："优化提示词"
人话："把话说清楚"

就像：
- "跟助理交代工作"
- "跟朋友说话"
```

### 【为什么这么多术语？】

**原因 1：技术人员写的**
```
- 技术人员习惯说术语
- 觉得大家都懂
- 其实小白不懂
```

**原因 2：显得专业**
```
- 用术语显得厉害
- 其实没必要
- 说人话更好
```

**原因 3：翻译问题**
```
- 国外技术引进来
- 直接翻译英文
- 没有本土化
```

**但这本手册：**
```
✅ 说人话
✅ 用比喻
✅ 不用术语
✅ 小白能懂
```

---

### 【如何绕过术语？】

**方法 1：问 AI**
```
遇到不懂的词：
直接问龙虾："XXX 是什么意思？"

示例：
你："什么是 Token？"
龙虾："就是字数，就像手机流量"
```

**方法 2：查术语表**
```
手册附录有术语表：
- 技术术语 → 人话
- 一看就懂
- 随时查
```

**方法 3：跳过术语】
```
- 不用每个词都懂
- 知道怎么用就行
- 就像用微信不用懂 4G/5G
```

**方法 4：找人问】
```
- 问子女
- 问朋友
- 问社群
- 别不好意思
```
