## 1.2 为什么叫"龙虾"？（Claw=爪子，中国社区文化）

**Q: 为什么叫"龙虾"？跟吃的有关系吗？A: 跟吃的没关系，是行业梗，Claw=爪子。**

---

### 【先说结论】

```
龙虾 = OpenClaw 的中文昵称

Claw = 爪子
OpenClaw = 开源爪子
中国社区 = 龙虾

就像：
- "猫" = Cat 的中文
- "狗" = Dog 的中文
- "龙虾" = Claw 的中文
```

### 【名字由来】

**第 1 步：OpenClaw 项目**
```
2026 年初：
- Peter Steinberger 创建项目
- 英文名：OpenClaw
- Claw 意思：爪子
- 寓意：帮你抓住重要的事
```

**第 2 步：中国社区引入**
```
2026 年 2 月：
- 中国开发者引入项目
- 直译："开源爪子"
- 不好听
- 有人提议叫"龙虾"
```

**第 3 步："龙虾"火了**
```
2026 年 3 月：
- "养龙虾" = 部署 AI 助理
- "你养龙虾了吗？"
- "我养了 3 只龙虾"
- 成了行业梗
```

**第 4 步：文化形成**
```
现在：
- 不只是项目名
- 是 AI 助理品类代名词
- "龙虾智能体"
- "龙虾生态"
```

### 【为什么是龙虾？】

**原因 1：Claw=爪子**
```
- 龙虾有爪子
- Claw 也是爪子
- 形象
- 好记
```

**原因 2：中国饮食文化**
```
- 龙虾是美食
- 大家都喜欢
- 亲切
- 吉利
```

**原因 3：好记好传播**
```
对比：
❌ "开源 AI 智能体框架"（太长）
❌ "OpenClaw"（英文，不好记）
✅ "龙虾"（两个字，好记）
```

**原因 4：社区文化**
```
- 开发者喜欢玩梗
- "养虾人"
- "虾农"
- "龙虾养殖基地"
- 有趣
- 有归属感
```

### 【龙虾文化】

**称呼：**
```
- 开发者 = "养虾人"
- 用户 = "虾农"
- 部署 = "养虾"
- 技能市场 = "虾池"
- 社区 = "龙虾养殖基地"
```

**黑话：**
```
- "你养龙虾了吗？" = 你部署 AI 助理了吗？
- "我养了 3 只" = 我部署了 3 个 AI 助理
- "我的龙虾很聪明" = 我的 AI 助理很好用
- "今天喂龙虾了吗？" = 今天用 AI 助理了吗？
```

**梗图：**
```
- 龙虾戴眼镜 = 智能龙虾
- 龙虾拿电脑 = 工作龙虾
- 龙虾睡觉 = 离线龙虾
- 龙虾打架 = 多助理协作
```
