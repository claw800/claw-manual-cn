# 一、认知篇 — 什么是龙虾

---

## 1.1 龙虾是什么？（说人话版）

**Q: 龙虾到底是什么？不是吃的吗？A: 不是吃的，是你的 AI 助理/伙伴。**

---

### 【先说结论】

```
龙虾 = AI 助理/伙伴

不是吃的，
是帮你干活的数字伙伴。

就像：
- 真人助理：帮你查资料/写东西/安排日程
- 龙虾助理：帮你查资料/写东西/安排日程
- 区别：龙虾 24 小时在线，不休息
```

---

### 【龙虾能干什么？】

**三类工作：1. 查资料**
```
✅ 查政策法规
✅ 查行业报告
✅ 查竞品信息
✅ 查学习资料
✅ 查新闻热点

就像：
- 有个研究员帮你查
- 查完总结好给你
- 不用你自己搜
```

**2. 写内容**
```
✅ 写文案（商品/朋友圈/小红书）
✅ 写报告（日报/周报/会议纪要）
✅ 写邮件（客户沟通/合作邀约）
✅ 写方案（活动策划/营销计划）
✅ 写脚本（短视频/直播话术）

就像：
- 有个文案帮你写
- 写完你修改
- 效率高
```

**3. 做任务**
```
✅ 整理数据（Excel 汇总）
✅ 文件管理（分类/重命名）
✅ 定时提醒（会议/缴费/生日）
✅ 流程自动化（订单处理/客户跟进）

就像：
- 有个秘书帮你做
- 你说一句话
- 它自动完成
```

---

### 【龙虾的特点】

**特点 1：24 小时在线**
```
✅ 随时问
✅ 秒回复
✅ 不休息
✅ 不嫌烦
```

**特点 2：有记忆**
```
✅ 记得你的名字
✅ 记得你的职业
✅ 记得你的偏好
✅ 越用越懂你
```

**特点 3：能干活**
```
✅ 不只是聊天
✅ 能执行任务
✅ 能操作文件
✅ 能连接其他工具
```

**特点 4：本地部署**
```
✅ 数据在你电脑
✅ 只有你能看
✅ 安全
✅ 隐私
```

```
└──────────────────────────────────────────────────┘
```

### 【真实案例】

**案例 1：淘宝店主**
```
用龙虾前：
- 每天花 3 小时写文案
- 客服回复 200+ 咨询
- 订单手动录入

用龙虾后：
- 30 分钟修改 AI 文案
- AI 自动回复 80% 咨询
- 订单自动整理

原话：
"龙虾就是我的 24 小时助理，
还不拿工资。"
```

**案例 2：内容创作者**
```
用龙虾前：
- 每周写 3 篇公众号
- 每篇 3 小时
- 粉丝 1 万

用龙虾后：
- 每周写 5 篇（AI 初稿 + 人修改）
- 每篇 1 小时
- 粉丝 3 万

原话：
"龙虾帮我写初稿，
我负责创意和修改。"
```

**案例 3：家长**
```
用龙虾前：
- 到处找小升初真题
- 孩子错题手动整理
- 不知道薄弱点在哪

用龙虾后：
- AI 自动出题（按知识点）
- 错题自动整理成册
- 薄弱点清晰

原话：
"龙虾就是孩子的 AI 老师，
随时在线。"
```

### 【一句话总结】

```
龙虾 = 你的 24 小时数字助理

能：
- 查资料
- 写内容
- 做任务

特点：
- 24 小时在线
- 有记忆
- 能干活
- 本地部署

核心：
不是"取代你"，
是"帮你"。
```
