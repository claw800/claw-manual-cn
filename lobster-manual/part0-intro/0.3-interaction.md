## 0.3 交互方式：就是聊天

**Q: 我怎么跟龙虾"说话"？复杂吗？**

**A: 就像微信聊天一样简单。**

---

### 【先说结论】

```
❌ 不是：
- "发布指令"（太正式）
- "写代码"（太复杂）
- "配置参数"（太技术）

✅ 是：
- "聊天"（就像微信）
- "发消息"（自然语言）
- "提需求"（说人话）

核心：
你会用微信，
就会用龙虾。
```

### 【怎么聊？】

**示例 1：打招呼**
```
你："你好"
龙虾："你好！我是你的龙虾助理，
       有什么可以帮你的？"
```

**示例 2：查资料**
```
你："帮我查一下上海公办初中入学条件"
龙虾："好的，已查好：
       1. 居住证（父母一方）
       2. 积分≥120 分
       3. 社保连续缴纳
       4. 实际居住证明
       具体政策文件链接：xxx"
```

**示例 3：写东西**
```
你："帮我写个淘宝商品标题，卖儿童学习桌"
龙虾："已写好 5 个版本：
       1. 【护脊神器】儿童学习桌可升降...
       2. 小学生写字桌实木护眼...
       3. ...
       推荐第 2 个，点击率高"
```

**示例 4：做任务**
```
你："把这 100 个订单信息整理成 Excel"
龙虾："好的，已整理好：
       - 订单号
       - 客户姓名
       - 商品名称
       - 金额
       - 发货状态
       文件已保存到：xxx.xlsx"
```

---

### 【聊天特点】

**特点 1：自然语言**
```
✅ 说人话
✅ 不用专业术语
✅ 就像跟助理说话
```

**特点 2：多轮对话**
```
你："帮我写个文案"
龙虾："好的，什么产品？"
你："儿童学习桌"
龙虾："有什么卖点？"
你："护脊/可升降/实木"
龙虾："好的，已写好..."
```

**特点 3：记住你**
```
第 1 次：
你："我叫张三，在上海做电商"

第 10 次：
你："帮我写个文案"
龙虾："好的，还是儿童学习桌吗？
       上次那篇转化率不错"
```

**特点 4：随时在线**
```
✅ 24 小时在线
✅ 随时问
✅ 秒回复
✅ 不嫌烦
```

---

### 【和微信聊天的区别】

**相同点：**
```
✅ 都是发消息
✅ 都是文字交流
✅ 都能发文件
✅ 都简单
```

**不同点：**
```
微信：
- 跟人聊天
- 对方可能忙
- 对方可能不回
- 对方可能不懂

龙虾：
- 跟 AI 聊天
- 24 小时在线
- 秒回复
- 什么都懂（几乎）
```

---

### 【和豆包/ChatGPT 的区别】

**豆包/ChatGPT：**
```
- 聊天为主
- 每次对话独立
- 记不住你
- 用完就忘
```

**龙虾：**
```
- 能干活（执行任务）
- 长期记忆
- 记得住你
- 越用越懂你
```

**比喻：**
```
豆包/ChatGPT = 功能手机
- 能打电话
- 基础功能

龙虾 = 智能手机
- 能打电话
- 还能做很多事
- 有记忆
- 能安装 APP（技能）
```
