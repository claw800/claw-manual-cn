## 0.2 核心隐喻："养龙虾"是什么？

**Q: 为什么叫"养龙虾"？龙虾是什么？**

**A: 这是一个行业梗，Claw=爪子，帮你抓住重要的事。**

---

### 【先说结论】

```
"养龙虾" = 部署 AI 助理

就像：
- "养猫" = 照顾猫咪
- "养狗" = 照顾狗狗
- "养龙虾" = 部署 AI 助理

核心：
不是吃的，
是帮你干活的数字伙伴。
```

---

### 【为什么叫"龙虾"？】

**原因 1：Claw=爪子**
```
OpenClaw → 开源爪子 → 龙虾

Claw 的意思：
- 爪子
- 帮你抓住重要的事
- 像助理的手

所以：
- OpenClaw = 开源龙虾
- 养龙虾 = 部署 AI 助理
```

**原因 2：中国社区文化**
```
- 国外叫"AI Agent"
- 中国叫"龙虾"
- 更亲切
- 更好记

就像：
- "猫"比"猫科动物"亲切
- "狗"比"犬科动物"好记
- "龙虾"比"AI Agent"好记
```

**原因 3：行业梗**
```
- 开发者先开始叫
- 用户跟着叫
- 越传越广
- 成了文化

现在：
- "你养龙虾了吗？"
- "我养了 3 只龙虾"
- "我的龙虾很聪明"
```

---

### 【"养"是什么意思？】

**养 = 部署 + 使用 + 培养**

**部署：**
```
- 安装龙虾
- 配置环境
- 连接飞书
- 开始使用
```

**使用：**
```
- 每天聊天
- 让它干活
- 查资料/写东西/做任务
```

**培养：**
```
- 告诉它你的信息
- 它越来越懂你
- 像养宠物一样
- 越养越聪明
```

---

### 【养龙虾 vs 养宠物】

```
┌──────────────────────────────────────────────────┐
│          养龙虾 vs 养宠物对比                     │
├──────────────────────────────────────────────────┤
│ 维度        │ 养龙虾          │ 养宠物          │
├──────────────────────────────────────────────────┤
│ 需要喂食    │ 不用（用电）     │ 需要（粮食）     │
│ 需要遛      │ 不用（在线）     │ 需要（出门）     │
│ 会掉毛      │ 不会            │ 会              │
│ 会生病      │ 不会（会出 bug） │ 会              │
│ 陪伴你      │ 会（24 小时）     │ 会              │
│ 帮你干活    │ 会（查/写/做）   │ 不会（只会卖萌） │
│ 越养越聪明  │ 会（记忆）       │ 会（训练）       │
└──────────────────────────────────────────────────┘
```

---

### 【养几只龙虾合适？】

**新手：1 只**
```
- 先养 1 只试试
- 熟悉了再说
- 别贪多
```

**进阶：2-3 只**
```
- 工作用 1 只
- 生活用 1 只
- 学习用 1 只
- 分工明确
```

**高手：多只**
```
- 不同场景用不同龙虾
- 不同技能用不同龙虾
- 像团队一样
```

**建议：**
```
新手从 1 只开始，
别一下养太多。
```

---

### 【龙虾吃什么？】

**吃电：**
```
- 本地部署：电费 ¥30-50/月
- 云端部署：Token 费用（可选）
- 不贵
```

**吃数据：**
```
- 你告诉它的信息
- 你的使用习惯
- 你的偏好
- 越吃越聪明
```

**不吃：**
```
❌ 不吃粮食
❌ 不吃水
❌ 不吃零食
```
