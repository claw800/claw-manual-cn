### 【手册结构】

```
 前置知识（入门必读）
→ 新手先看这部分（30 分钟）

一、认知篇（什么是龙虾）
→ 想了解龙虾时看

二、价值篇（能干什么/值不值）
→ 最关心，优先看

三、信任篇（安全/隐私/焦虑）
→ 有担忧时看

四、使用篇（怎么用/怎么部署）
→ 最关心，优先看

→ 进阶后看

附录（索引/术语/路径）
→ 随时查
```

### 【如何快速查找？】

**方法 1：看目录**
```
- 手册有详细目录
- 找相关章节
- 直接跳过去
```

**方法 2：用索引**
```
- 附录有问题索引
- 按问题找答案
- 快速定位
```

**方法 3：问 AI**
```
直接问龙虾：
"XXX 怎么办？"
"手册里哪章讲这个？"

龙虾会告诉你：
- 答案
- 相关章节
- 页码
```

**方法 4：搜索**
```
电子版：
- Ctrl+F 搜索关键词
- 快速找到
- 方便
```
