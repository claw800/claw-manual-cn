## 4.2 硬件要求：什么电脑能用？

**Q: 我的老电脑能用吗？需要买新设备吗？手机能用吗？A: 近 5 年的电脑基本都能用，手机也能用。**

---

### 【先说结论】

```
❌ 不需要：
- 买新电脑（老电脑能用）
- 买服务器（家用电脑就行）
- 买高端设备（中低配够用）

✅ 需要：
- 近 5 年的电脑
- 8G 以上内存
- 50G 硬盘空间
```

---

### 【最低配置】

**能跑起来的底线：**
```
CPU：Intel i5 / AMD Ryzen 5（近 5 年）
内存：8GB（最低）
硬盘：50GB 剩余空间（SSD 推荐）
系统：Windows 10 / macOS 11 / Linux
网络：稳定宽带
```

**能做什么：**
```
✅ 基础聊天
✅ 简单文案
✅ 资料查询
✅ 文件整理
```

**限制：**
```
❌ 大模型跑不动（会卡）
❌ 多任务处理慢
❌ 复杂任务吃力
```

**适合：**
```
- 预算有限
- 轻度使用
- 先试试水
```

---

**流畅使用的标准：**
```
CPU：Intel i7 / AMD Ryzen 7（近 3 年）
内存：16GB（推荐）
硬盘：100GB 剩余空间（SSD 必须）
系统：Windows 11 / macOS 12+ / Linux
网络：稳定宽带（50M+）
```

**能做什么：**
```
✅ 所有基础功能
✅ 中等模型流畅
✅ 多任务处理
✅ 复杂任务
```

**限制：**
```
❌ 超大模型还是卡
❌ 同时跑多个 AI 吃力
```

**适合：**
```
- 日常使用
- 工作依赖
- 长期使用
```

### 【老电脑能用吗？】

**判断标准：**
```
✅ 能用：
- 2019 年后的电脑
- 内存 8G 以上
- 能正常上网看视频

❌ 吃力：
- 2016 年前的电脑
- 内存 4G 以下
- 机械硬盘 + 老 CPU
```

**优化方案：**
```
方案 1：加内存
- 8G → 16G（¥200-300）
- 提升明显

方案 2：换 SSD
- 机械 → 固态（¥300-500）
- 速度提升 5 倍

方案 3：用小模型
- Gemma 4 Nano（2GB）
- 不占资源

方案 4：用云端
- 本地只负责交互
- 计算在云端
- 老电脑也能用
```

**真实案例：**
```
案例 1：2018 年 MacBook Pro
- 内存 8G
- 硬盘 256G
- 结果：流畅使用（小模型）

案例 2：2017 年 Windows 笔记本
- 内存 4G
- 机械硬盘
- 结果：加内存到 8G + 换 SSD，流畅

案例 3：2015 年老台式机
- 内存 4G
- 老 CPU
- 结果：用云端 API，流畅
```
