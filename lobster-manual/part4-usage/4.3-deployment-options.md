## 4.3 部署方案：本地 vs 云端

**Q: 有几种部署方式？我该选哪种？**

**A: 3 种主流方案，看你的需求选。**

---

### 【先说结论】

```
3 种方案：

方案 A：云端版（一键部署）
- 最快（2 分钟）
- 最简单（点鼠标）
- 适合新手体验

方案 B：本地版（自己部署）
- 最安全（数据在手）
- 最便宜（长期）
- 适合长期使用

方案 C：混合版（端云协同）
- 最灵活（按需切换）
- 最平衡（安全 + 能力）
- 适合进阶用户
```

---

### 【方案 A：云端版】

**是什么：**
```
- 龙虾跑在云端服务器上
- 你通过飞书访问
- 数据存在云端
```

**优点：**
```
✅ 一键部署（2 分钟完成）
✅ 零配置（不用懂技术）
✅ 限时免费（400 万 Token）
✅ 不占本地资源（老电脑能用）
✅ 随时随地用（有网就行）
```

**缺点：**
```
❌ 数据在云端（隐私担忧）
❌ Token 有限（用完付费）
❌ 功能受限（不能用高级技能）
❌ 依赖网络（没网不能用）
❌ 长期贵（¥99/月）
```

**成本：**
```
- 首次：¥0
- 长期：¥0-99/月
```

**适合：**
```
✅ 新手想先试试
✅ 完全不懂技术
✅ 轻度使用
✅ 老电脑用户
```

**部署步骤：**
```
1. 打开部署页面
2. 点击"一键部署 OpenClaw"
3. 授权飞书账号
4. 等待自动安装（2 分钟）
5. 开始使用
```

---

### 【方案 B：本地版】

**是什么：**
```
- 龙虾跑在你自己电脑上
- 数据存在本地硬盘
- 完全由你控制
```

**优点：**
```
✅ 数据本地（最安全）
✅ 长期便宜（模型免费）
✅ 功能完整（所有技能）
✅ 不依赖网络（离线能用）
✅ 完全控制（想怎么用都行）
```

**缺点：**
```
❌ 需要学习（看教程）
❌ 需要时间（2-4 小时部署）
❌ 占本地资源（电脑要有一定配置）
❌ 自己维护（出问题自己解决）
```

**成本：**
```
- 首次：¥0（已有电脑）
- 长期：¥0-50/月（电费）
```

**适合：**
```
✅ 有点技术基础
✅ 重视隐私安全
✅ 长期使用
✅ 重度用户
```

**部署步骤：**
```
1. 下载安装包（官网）
2. 运行安装程序
3. 配置模型（选 Gemma 4）
4. 配置飞书连接
5. 启动服务
6. 开始使用

时间：30-60 分钟（第一次）
```

---

### 【方案 C：混合版（端云协同）】

**是什么：**
```
- 日常用本地模型（免费 + 安全）
- 复杂任务用云端 API（能力强）
- 自动切换（智能调度）
```

**优点：**
```
✅ 平衡安全 + 能力
✅ 日常免费（本地模型）
✅ 复杂任务强（云端 API）
✅ 灵活控制（按需切换）
✅ 成本可控（云端按需付费）
```

**缺点：**
```
❌ 配置复杂（要设两套）
❌ 需要判断（什么时候用云端）
❌ 成本不确定（看使用量）
```

**成本：**
```
- 首次：¥0
- 长期：¥50-200/月（云端 API）
```

**适合：**
```
✅ 进阶用户
✅ 既想要安全又想要能力
✅ 有复杂任务需求
✅ 愿意折腾
```

**配置示例：**
```
日常任务 → 本地 Gemma 4
- 聊天
- 简单文案
- 资料查询

复杂任务 → 云端 Claude/GPT-4
- 长文档写作
- 代码生成
- 专业分析

自动规则：
- Token<1000 → 本地
- Token>1000 → 云端
- 需要联网 → 云端
```

### 【真实案例】

**案例 1：淘宝店主（义乌）**
```
选择：本地版

原因：
- 订单数据敏感（不能外泄）
- 每天都要用（长期便宜）
- 有技术基础（自己搞定）

结果：
- 部署 1 小时完成
- 数据安全放心
- 每月省¥100 云端费
```

**案例 2：内容创作者（上海）**
```
选择：混合版

原因：
- 日常写作本地够用
- 复杂方案需要云端
- 成本要控制

结果：
- 80% 任务本地（免费）
- 20% 任务云端（¥50/月）
- 平衡最好
```

**案例 3：家长（上海）**
```
选择：云端版

原因：
- 完全不懂技术
- 只是试试水
- 孩子教育用（数据不敏感）

结果：
- 2 分钟部署完成
- 免费额度够用
- 确定有用再升级
```

### 【选择建议】

**决策树：**
```
Q1：懂技术吗？
├─ 不懂 → 云端版
└─ 懂 → Q2

Q2：重视隐私吗？
├─ 不重视 → 云端版
└─ 重视 → Q3

Q3：长期使用吗？
├─ 试试水 → 云端版
├─ 长期用 → 本地版
└─ 都要 → 混合版
```

**一句话建议：**
```
新手：先云端（试试水）
确定：转本地（长期用）
进阶：用混合（最灵活）
```

---

### 【常见问题】

**Q1：能从云端转本地吗？**
```
A：可以。
- 数据可以导出
- 配置可以迁移
- 随时切换
```

**Q2：本地部署难吗？**
```
A：有教程。
- 官网有视频
- 社群有人问
- 实在不行找上门（¥300-800）
```

**Q3：混合版怎么切换？**
```
A：自动或手动。
- 自动：按规则切换
- 手动：命令指定
- 默认：优先本地
```

---

### 【一句话总结】

```
没有最好的方案，
只有最适合你的。

建议：
- 新手：云端版（先试试）
- 长期：本地版（最划算）
- 进阶：混合版（最灵活）

核心：
先动起来，
后面可以换。
```
