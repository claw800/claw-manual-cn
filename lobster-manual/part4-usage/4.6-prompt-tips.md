## 4.6 提示词技巧：让 AI 输出翻倍

**Q: 怎么跟龙虾说话，它才能更好理解我的意思？A: 记住一个核心：把它当真人，说人话。**

---

### 【先说结论】

```
❌ 错误方式：
- 像下命令（"写文案"）
- 太简短（"总结"）
- 没上下文（"这个怎么样"）

✅ 正确方式：
- 像跟助理说话（"帮我写个文案"）
- 给背景信息（"淘宝卖儿童学习桌"）
- 说清楚要求（"简洁风格，200 字以内"）

核心：
把它当真人助理，
你说得越清楚，
它做得越好。
```

---

### 【基础公式】

**好提示词 = 角色 + 任务 + 背景 + 要求示例 1：写文案**
```
❌ 差："写文案"
✅ 好："你是淘宝文案专家，帮我写个儿童学习桌的商品标题，
 卖点是护脊/可升降/实木，
 风格简洁，20 字以内，写 5 个版本"
```

**示例 2：查资料**
```
❌ 差："查政策"
✅ 好："你是政策研究员，帮我查上海公办初中入学条件，
 我是非沪籍家庭，居住证积分达标，
 需要具体条件和材料清单，
 附上官方文件链接"
```

**示例 3：整理数据**
```
❌ 差："整理 Excel"
✅ 好："你是数据分析师，把这 100 个订单整理成 Excel，
 包含：订单号/客户姓名/商品/金额/状态，
 按金额排序，
 标出超过 1000 元的订单"
```

---

### 【进阶技巧】

**技巧 1：给角色**
```
"你是 XXX 专家..."

示例：
- "你是资深文案..."
- "你是数据分析师..."
- "你是教育顾问..."
- "你是法律顾问..."

作用：让它进入角色，输出更专业。
```

**技巧 2：给背景**
```
"背景是 XXX..."

示例：
- "我是淘宝卖家，卖儿童学习桌"
- "我女儿五年级，数学好，英语弱"
- "我们公司做 AI 培训，面向小白"

作用：让它了解你，输出更精准。
```

**技巧 3：给示例**
```
"像这样..."

示例：
- "标题风格像这个：【护脊神器】儿童学习桌..."
- "报告格式参考附件"
- "用上次那种简洁风格"

作用：让它有参照，输出更符合预期。
```

**技巧 4：给限制**
```
"要求 XXX..."

示例：
- "200 字以内"
- "用表格形式"
- "分 3 个部分"
- "不要专业术语"

作用：让它有边界，输出更可控。
```

**技巧 5：分步骤**
```
"第一步 XXX，第二步 XXX..."

示例：
- "先查资料，再总结，最后给建议"
- "先分析数据，再生成图表，最后写结论"

作用：复杂任务拆解，输出更有逻辑。
```

### 【场景示例】

**场景 1：电商文案**
```
❌ 差："写个标题"

✅ 好：
"你是淘宝文案专家，帮我写儿童学习桌标题。
产品信息：
- 卖点：护脊/可升降/实木/E0 环保
- 目标用户：小学生家长
- 价格区间：800-1200 元
- 竞品标题参考：【护脊神器】...

要求：
- 写 5 个版本
- 每个 20 字以内
- 突出差异化卖点
- 适合直通车投放"
```

**场景 2：工作汇报**
```
❌ 差："写日报"

✅ 好：
"你是行政助理，帮我写今天的工作日报。
今天做了：
- 完成客户 A 的方案（已发送）
- 跟进客户 B 的合同（等回复）
- 参加团队会议（讨论 Q2 目标）
- 学习龙虾使用（部署完成）

要求：
- 分 3 部分：完成/进行中/计划
- 简洁专业
- 突出成果
- 200 字以内"
```

**场景 3：孩子学习**
```
❌ 差："出题"

✅ 好：
"你是小学数学老师，给我女儿出应用题。
学生情况：
- 五年级
- 数学好，计算强
- 薄弱点：行程问题
- 目标：小升初

要求：
- 出 10 道行程问题
- 难度中等偏上
- 附答案解析
- 格式适合打印"
```

**场景 4：市场调研**
```
❌ 差："查竞品"

✅ 好：
"你是市场分析师，帮我调研竞品。
我的产品：
- AI 培训课，面向小白
- 价格：¥2999
- 形式：线上 + 线下

调研要求：
- 找 5 个主要竞品
- 分析价格/内容/形式
- 找出差异化机会
- 附竞品官网链接
- 用表格对比"
```

### 【避坑指南】

**避坑 1：太简短**
```
❌ "写文案"
✅ "帮我写个淘宝商品文案，卖儿童学习桌..."

原因：信息太少，它不知道写什么。
```

**避坑 2：没背景**
```
❌ "这个怎么样"
✅ "我写了个文案，帮我看看怎么样..."

原因：没有上下文，它不知道"这个"是什么。
```

**避坑 3：要求模糊**
```
❌ "写好点"
✅ "用简洁风格，200 字以内，突出卖点"

原因："好"是主观的，要说具体标准。
```

**避坑 4：一次太多**
```
❌ "写文案 + 查资料 + 整理数据 + 做报表"
✅ "先写文案，写完我再让你做别的"

原因：任务太多，容易乱。分步来。
```

**避坑 5：不检查**
```
❌ 直接用它输出的
✅ 先检查再使用

原因：AI 可能出错，重要内容要检查。
```

---

### 【高级技巧】

**技巧 1：让它思考**
```
"请一步步思考..."

示例：
"请一步步思考：
1. 先分析用户需求
2. 再确定文案方向
3. 然后写标题
4. 最后优化"

作用：让它有逻辑，输出更可靠。
```

**技巧 2：让它自查**
```
"写完后检查..."

示例：
"写完后检查：
- 有没有错别字
- 数据是否准确
- 逻辑是否通顺
- 是否符合要求"

作用：减少错误，提高质量。
```

**技巧 3：让它追问**
```
"如果信息不够，问我..."

示例：
"帮我写方案，如果信息不够，先问我。"

作用：避免它瞎猜，输出更准确。
```

**技巧 4：多轮对话**
```
第 1 轮："帮我写个文案"
第 2 轮："第 2 个版本不错，再优化一下..."
第 3 轮："太长了，精简到 20 字"

作用：逐步逼近最佳结果。
```
