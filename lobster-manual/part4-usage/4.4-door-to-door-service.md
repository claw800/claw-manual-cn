### 【预约流程】

**步骤 1：提交需求**
```
渠道：
- 官网预约页面
- 客服电话
- 微信客服
- 社群推荐

信息：
- 城市 + 区域
- 期望时间
- 服务类型（基础/标准/高级）
- 电脑情况（系统/配置）
```

**步骤 2：确认报价**
```
服务商联系你：
- 确认具体需求
- 报价（含远程费如有）
- 确认时间
- 发送服务清单
```

**步骤 3：等待上门**
```
准备：
✅ 电脑充满电（或插电）
✅ 网络正常
✅ 飞书账号注册好
✅ 问题清单（想问的）
```

**步骤 4：现场服务**
```
流程：
1. 工程师上门（带工牌）
2. 确认需求（10 分钟）
3. 开始部署（1-2 小时）
4. 功能测试（20 分钟）
5. 使用培训（30 分钟）
6. 问题解答（20 分钟）
7. 签字确认
```

**步骤 5：售后服务**
```
包含：
✅ 7 天远程支持（微信/电话）
✅ 30 天内免费调整 1 次
✅ 使用问题随时问
```

### 【需要配合什么？】

**硬件准备：**
```
✅ 电脑（Windows/Mac）
✅ 电源（插电）
✅ 网络（稳定宽带）
✅ 飞书账号（提前注册）
```

**软件准备：**
```
✅ 管理员权限（安装需要）
✅ 杀毒软件暂时关闭（可能误报）
✅ 关闭其他程序（省资源）
```

**信息准备：**
```
✅ 你的基本信息（姓名/职业）
✅ 使用场景（电商/办公/教育）
✅ 想实现的功能清单
✅ 问题清单（不懂的）
```

**时间配合：**
```
✅ 预留 2-3 小时
✅ 全程参与（边学边问）
✅ 最好有纸笔（记要点）
```

### 【避坑指南】

**避坑 1：选择正规服务商**
```
✅ 看资质（官网认证）
✅ 看评价（社群问）
✅ 看案例（真实客户）
❌ 不要贪便宜（¥100 那种）
```

**避坑 2：确认服务清单**
```
✅ 提前确认包含什么
✅ 确认不包含什么
✅ 确认售后支持
❌ 不要口头约定（要书面）
```

**避坑 3：现场学习**
```
✅ 全程参与（不要走开）
✅ 多问问题（当场搞懂）
✅ 记笔记（回去复习）
❌ 不要完全依赖（自己要学会）
```

**避坑 4：验收测试**
```
✅ 现场测试核心功能
✅ 确认能用再签字
✅ 留联系方式（售后）
❌ 不要着急签字
```
