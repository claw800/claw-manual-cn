### 【避坑指南】

**避坑 1：别买太贵的课**
```
❌ 避免：
- "AI 赚钱课"（¥9999+）
- "躺赚秘籍"（不靠谱）
- "内部消息"（骗子）

✅ 建议：
- 先免费学（B 站/文档）
- 确定需要再买课（¥299-699）
- 看评价/试听
```

**避坑 2：别信"包教包会"**
```
❌ 避免：
- "3 天精通"（不可能）
- "包教包会"（看自己）
- "学不会退款"（套路）

✅ 建议：
- 学习需要时间（2-4 周）
- 关键是自己练
- 没有捷径
```

**避坑 3：别盲目考证**
```
❌ 避免：
- 刚学完就考证（没用）
- 为考证而考证（浪费钱）
- 迷信证书（能力重要）

✅ 建议：
- 先用起来
- 需要再考（接单/求职）
- 能力>证书
```
