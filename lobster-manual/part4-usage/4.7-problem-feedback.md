## 4.7 问题反馈：遇到问题先问龙虾

**Q: 遇到问题怎么办？怎么反馈？A: 先问龙虾自己，它比你想象的更懂自己。**

---

### 【先说结论】

```
遇到问题：

第 1 步：问龙虾（80% 问题能解决）
第 2 步：查教程（官网/社群）
第 3 步：问社群（QQ 群/微信群）
第 4 步：找客服（最后手段）

核心：
先问龙虾，
它是最懂自己的。
```

---

### 【为什么先问龙虾？】

**原因 1：它知道自己的局限**
```
你问："这个功能有吗？"
它会告诉你："这个功能还没有，但可以用 XXX 替代"
```

**原因 2：它能提供替代方案**
```
你问："PPT 怎么配图？"
它会说："我做不了，但可以用 Canva/稿定设计"
```

**原因 3：它能推荐工具/方法**
```
你问："这个报错怎么办？"
它会说："这是 XXX 问题，试试 XXX 方法"
```

**原因 4：比你查教程快**
```
你查：百度 → 找教程 → 看半天 → 还不一定对
你问：龙虾 → 10 秒 → 直接给答案
```

---

### 【真实案例】

**案例 1：PPT 没有图片**
```
用户："试了一个 PPT，色彩图片没有。"
龙虾："我能做 PPT 内容和结构，
 但配图需要用专门工具。
 推荐：
 1. Canva（在线设计）
 2. 稿定设计（模板多）
 3. Unsplash（免费图片）
 要我帮你整理 PPT 内容吗？"

结果：用户用 Canva 配图，问题解决。
```

**案例 2：安装失败**
```
用户："安装失败，报错 XXX"
龙虾："这个错误是网络问题。
 试试：
 1. 检查网络连接
 2. 关闭防火墙
 3. 用国内镜像源
 还不行可以找上门服务：¥300-800"

结果：用户换镜像源，安装成功。
```

**案例 3：配置不会弄**
```
用户："API Key 怎么填？"
龙虾："API Key 在你注册的平台上。
 步骤：
 1. 登录通义千问官网
 2. 进入控制台
 3. 找到 API 管理
 4. 创建 Key
 5. 复制到这里
 要我一步步带你操作吗？"

结果：用户跟着步骤，配置成功。
```

---

### 【怎么问龙虾？】

**问法 1：直接描述问题**
```
"我遇到了 XXX 问题，怎么办？"

示例：
- "安装失败，报错 XXX"
- "配置不会弄，API Key 在哪？"
- "PPT 没有图片，怎么配？"
```

**问法 2：问能不能做**
```
"你能做 XXX 吗？"

示例：
- "你能做 PPT 吗？"
- "你能直接发微信吗？"
- "你能打电话吗？"
```

**问法 3：问替代方案**
```
"XXX 做不了，有什么替代方法？"

示例：
- "配图做不了，有什么工具推荐？"
- "这个功能没有，怎么实现？"
```

**问法 4：问步骤**
```
"XXX 怎么做？一步步教我"

示例：
- "部署怎么做？一步步教我"
- "配置 API 怎么做？一步步教我"
```

### 【常见问题及问法】

**问题 1：安装类**
```
❌ 差："装不上"
✅ 好："安装失败，报错 XXX，怎么办？"

龙虾会问：
- 什么系统？
- 什么版本？
- 完整报错是什么？
```

**问题 2：配置类**
```
❌ 差："不会配置"
✅ 好："API Key 怎么配置？一步步教我"

龙虾会：
- 告诉你去哪找
- 一步步带你操作
- 有问题继续问
```

**问题 3：使用类**
```
❌ 差："不好用"
✅ 好："我想做 XXX，怎么做？"

龙虾会：
- 告诉你能不能做
- 如果能，教你怎么做
- 如果不能，推荐替代方案
```

**问题 4：故障类**
```
❌ 差："坏了"
✅ 好："XXX 功能用不了，报错 XXX"

龙虾会：
- 分析可能原因
- 给排查步骤
- 还不行推荐售后
```
