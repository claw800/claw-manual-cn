# 四、使用篇 — 怎么用/怎么部署

---

## 4.1 从零开始：怎么养？

**Q: 我是小白，从零开始怎么养龙虾？A: 4 步搞定，30 分钟上手。**

---

### 【先说结论】

```
养龙虾 = 拥有数字伙伴

不用懂技术，
按步骤来，
30 分钟就能用。
```

---

### 【第 1 步：准备环境】（5 分钟）

**你需要：**
```
✅ 一台电脑（Windows/Mac/Linux）
 - 近 5 年的电脑都能用
 - 内存 8G 以上（16G 推荐）
 - 硬盘有 50G 剩余空间

✅ 一个飞书账号（个人版免费）
 - 手机号注册
 - 电脑/手机都能登录

✅ 稳定的网络
 - 家用宽带即可
 - 不需要特殊网络
```

**不需要：**
```
❌ 不需要买新电脑（老电脑能用）
❌ 不需要懂编程（点鼠标就行）
❌ 不需要特殊网络（正常上网）
```

---

### 【第 2 步：选择部署方式】（5 分钟）

**3 种方式，选适合你的：方式 A：一键部署（推荐新手）**
```
- 点击按钮 → 自动安装
- 2 分钟完成
- 限时免费体验
- 适合：想先试试

操作：
1. 打开部署页面
2. 点击"一键部署"
3. 授权飞书账号
4. 等待完成
```

**方式 B：本地部署（推荐长期）**
```
- 按教程安装
- 30 分钟完成
- 数据在自己手里
- 适合：想长期使用

操作：
1. 下载安装包
2. 运行安装程序
3. 按提示配置
4. 完成启动
```

**方式 C：上门服务（推荐小白）**
```
- 专人上门安装
- 2 小时完成
- 包教包会
- 适合：完全不懂技术

操作：
1. 预约服务商
2. 等待上门
3. 现场教学
4. 开始使用
```

---

### 【第 3 步：配置模型】（10 分钟）

**选择 AI 大脑：选项 A：本地模型（免费）**
```
- Gemma 4（谷歌开源）
- 0 元使用
- 数据不出本地
- 适合：日常使用

配置：
1. 选择"本地模型"
2. 自动下载（约 10GB）
3. 等待完成
4. 开始使用
```

**选项 B：云端 API（付费）**
```
- 通义千问/文心一言/Claude
- ¥0.01-0.1/千 Token
- 能力更强
- 适合：复杂任务

配置：
1. 选择"云端 API"
2. 注册对应平台
3. 获取 API Key
4. 填入配置
```

**建议：**
```
新手：先用本地模型（免费）
进阶：本地 + 云端混合（按需）
```

---

### 【第 4 步：开始使用】（10 分钟）

**第一次对话：**
```
1. 打开飞书
2. 找到"龙虾助手"
3. 发送第一条消息

试试这些：
- "你好，我叫张三"
- "帮我写个日报"
- "查一下上海天气"
```

**设置记忆：**
```
告诉龙虾你的信息：
- "我在上海做电商"
- "我女儿叫 Zoe，五年级"
- "我喜欢简洁风格"

下次它会记住你。
```

**尝试技能：**
```
输入"/"查看可用技能：
- /写文案
- /查资料
- /整理数据
- /定提醒

选一个试试。
```

### 【常见问题】

**Q1：安装失败怎么办？**
```
A：
1. 检查网络（能正常上网吗？）
2. 检查空间（硬盘够吗？）
3. 重启电脑再试
4. 还不行就问龙虾："安装失败怎么办？"
```

**Q2：不会配置怎么办？**
```
A：
1. 看教程（官网有视频）
2. 问社群（QQ 群/微信群）
3. 找上门（¥300-800 元/次）
4. 还是不会？问龙虾："帮我配置"
```

**Q3：用起来卡怎么办？**
```
A：
1. 关闭其他程序（省内存）
2. 用小模型（Gemma 4 Nano）
3. 用云端 API（不占本地资源）
4. 考虑升级电脑
```
